# UI/UX 設計方針

## デザインテイスト

- **スタイル**: モダン・スタイリッシュ
- **カラーモード**: ダーク/ブックモード（ダーク系ベース）
- **フォント**: システムフォント（SF Pro 系統）
- **余白**: ゆったりとした余白でリッチな印象を持たせる

---

## 画面遷移フロー

```
ホーム画面（旅行一覧）
  │
  ├─[+ 旅行を追加]─→ 旅行作成画面
  │                       │
  │               [旅行を作成]
  │                       │
  └─[カードをクリック]─→ 旅行詳細画面
                          │
                 ┌────────┴────────┐
            [共有ボタン]      [スポット行クリック]
                 │                 │
           共有モーダル      スポット詳細モーダル
```

---

## カテゴリバッジのカラー定義

| カテゴリ | CSS カラートークン | 用途例 |
|---|---|---|
| 観光 | indigo 系 | 寺社・公園など |
| グルメ | amber 系 | 飲食店・市場など |
| 散策 | green 系 | 街歩き・自然散策など |
| 体験 | pink 系 | アクティビティ・乗り物体験など |
| 宿泊 | gray 系 | ホテル・旅館など |

バッジは小さなラベルとして各スポット行に表示する。背景色をカテゴリカラー、文字色は白またはダーク系で可読性を確保する。

---

## ドラッグ&ドロップのビジュアルフィードバック

| 状態 | 表現 |
|---|---|
| ドラッグ中の行 | 半透明（opacity 低下）表示 |
| ドロップ候補の位置 | 下線ハイライト（ボーダーまたはインジケーター） |
| ドロップ完了後 | 即時に並び順が更新される |

---

## インラインフォームパターン（スポット追加）

- スケジュールタブの「+ スポットを追加」ボタンを押すとリスト内にフォームが展開する
- 画面遷移はしない（インライン展開）
- フォームは入力項目をすべてインラインで持つ
- 「追加する」で登録・フォームを閉じる
- 「キャンセル」で入力内容を破棄・フォームを閉じる

---

## モーダルパターン

- **スポット詳細モーダル**: スポット行クリックで表示。背景クリックまたは「閉じる」ボタンで閉じる
- **共有モーダル**: 「共有」ボタンクリックで表示。背景クリックまたは「閉じる」ボタンで閉じる
- モーダルの背景はオーバーレイ（半透明の暗色）で下層のコンテンツをブロック

---

## タブナビゲーション

### 日付タブ（横スクロール対応）
- 「▶N日目・曜日（月/日）」形式
- 日数が多い場合は横スクロール
- アクティブタブはアンダーラインまたは背景色で強調

### 表示タブ（スケジュール / 地図）
- シンプルな2タブ切り替え
- アクティブタブはスタイルで区別

---

## 旅行カード（ホーム一覧）

各カードに以下を表示:
- 絵文字アイコン（大きく表示）
- 旅行タイトル
- 目的地
- 期間（YYYY/MM/DD 〜 YYYY/MM/DD）
- 日数

---

## 空状態（エンプティステート）

- 旅行が0件の場合: 旅行追加を促すメッセージとアイコンを表示
- スポットが0件の場合: スポット追加を促すメッセージを表示

---
